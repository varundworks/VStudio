# **App Name**: V Studio

## Core Features:

- Client Management: Allow users to input client details including name, address, and contact information.
- Invoice Creation: Enable the creation of invoices with fields for service descriptions, hours worked/flat rates, hourly rates/service rates, total calculation, and tax if applicable.
- Custom Template: Allows uploading a template of their choosing to allow incorporating user's branding.
- PDF Generation: Generate a PDF invoice based on the input data and the chosen template for the invoices.
- Invoice Delivery: Facilitate invoice delivery through direct download or sharing via email link.
- User Authentication: Include authentication via email/password, to identify legitimate users.
- Service Description Assistant: Use a generative AI tool to assist the user in constructing natural-language descriptions of the service performed and added to the invoice.

## Style Guidelines:

- Primary color: Deep sky blue (#3498DB) to inspire trust and reliability.
- Background color: Very light blue (#EBF5FB), a desaturated hue of the primary.
- Accent color: Vivid orange (#F39C12) for highlighting key actions.
- Headline font: 'Space Grotesk' (sans-serif), for a modern, clean look. Body font: 'Inter' (sans-serif).
- Use clean, outline-style icons for services and invoice actions.
- Maintain a responsive design adaptable to both desktop and mobile browsers.
- Use subtle animations and transitions to improve user experience (e.g., loading, confirmation).
